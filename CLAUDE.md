## O Que É Este Projeto

Estamos construindo uma **comunidade de Vibe Coding** focada em pessoas do mercado digital (não-programadores) que querem usar IA (Claude Code, Cursor, Lovable, etc.) para criar produtos, automações e negócios.

## Público-Alvo

- **"Devs de oferta"** — pessoas do mercado digital que não sabem programar
- Trabalham com tráfego, lançamento, criação de conteúdo
- Empresários que querem usar IA para criar coisas diferenciadas
- Perfil: sabem vender, falta a parte técnica — Vibe Coding resolve isso

## Modelo de Negócio

### Freemium (Gratuito + Pago)
- **Grátis**: Comunidade aberta com muito conteúdo de valor (funil de entrada)
- **Pago**: Comunidade premium, low ticket, seleciona quem quer ir mais fundo
- **Serviços**: Vender implementações/consultoria pra empresas (autoridade da comunidade)

### Monetização esperada
1. Assinatura da comunidade paga (preço a definir, faixa R$29-49/mês)
2. Implementações e consultoria para empresas
3. Possíveis infoprodutos futuros derivados da comunidade

## Stack Tecnológico

### Plataforma Central
- **Telegram**: Hub principal (canal broadcast + grupo de chat)
- Canal = conteúdo curado, novidades, links
- Grupo = troca entre membros, dúvidas, showcase

### Plataforma Complementar (a ser construída)
- Vibe-codada por nós mesmos
- Área de membros, projetos, tutoriais organizados
- Ferramentas exclusivas, automações, templates

### Ferramentas de Desenvolvimento
- Claude Code (Antigravity)
- Cursor
- Lovable (protótipos visuais)
- Supabase (backend/banco de dados)
- Stripe ou ASAAS (pagamentos)

## Dores da Audiência (Identificadas nos Comentários do Instagram)

1. **Segurança** — Medo de criar apps vulneráveis
2. **Curadoria de ferramentas** — Qual IA usar, qual stack, qual gateway de pagamento
3. **Passo a passo prático** — Querem replicar, não só ver
4. **Networking** — Builders precisam de lugar pra trocar ideia
5. **Custos/créditos** — Limites das IAs, como otimizar

## Conteúdo Planejado

- Lives semanais (build in public)
- Newsletter/atualizações via canal Telegram
- Interação diária no grupo
- Desafios e sprints
- Automações de curadoria (scraping Twitter/X → posts no canal)

## Ideias de Software Para Construir

- Diretório de projetos dos membros
- Curadoria automatizada de IA news

- Gamificação/ranking de contribuição
- Painel comparativo de ferramentas
- Área de membros (plataforma própria)
